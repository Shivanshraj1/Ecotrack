# EcoTrack

EcoTrack is a modern SaaS-style React dashboard for AI-powered waste segregation and recycling tracking.

## Stack
- React + Vite
- Tailwind CSS
- React Router
- Chart.js

## Run locally
```bash
npm install
npm run dev
```

## Routes
- `/` landing page
- `/login` login
- `/register` register
- `/dashboard` main dashboard
- `/log` log waste
- `/pickups` pickup requests
- `/rewards` rewards + leaderboard
- `/admin` admin panel
